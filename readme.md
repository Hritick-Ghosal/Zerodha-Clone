Zerodha Clone

A front-end clone of Zerodha, India’s most popular stock trading platform.
This project replicates the UI/UX of Zerodha’s landing page and core sections using clean HTML, CSS, and JavaScript , React and various Backend Technologies

 Features

Fully responsive landing page

Clean and minimal design inspired by Zerodha

Navbar with logo and navigation links

Hero section with CTA button

Pricing section

Product showcase section

Footer with essential links

Smooth UI with consistent spacing and typography
